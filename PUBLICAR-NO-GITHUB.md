# Publicação rápida no GitHub Pages

1. Entre no GitHub e crie o repositório `cine-guerreiro`.
2. Dentro do repositório, escolha **Add file > Upload files**.
3. Envie todos os arquivos e pastas deste projeto, incluindo a pasta oculta `.github`.
4. Confirme o envio para a branch `main`.
5. Abra **Settings > Pages**.
6. Em **Source**, selecione **GitHub Actions**.
7. Abra a aba **Actions** e acompanhe o workflow **Publicar Angular no GitHub Pages**.
8. Ao finalizar, acesse `https://capf01.github.io/cine-guerreiro/`.

Caso o repositório tenha outro nome, o workflow ajustará automaticamente o endereço-base do Angular.
